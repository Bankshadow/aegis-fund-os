## 1. วิเคราะห์เปรียบเทียบโทนภาษา

### โทนกรุงไทย (ต้นแบบ)
| ลักษณะ | ตัวอย่างจริงจากเว็บกรุงไทย |
|---|---|
| ยืนเคียงข้าง ไม่สั่งสอน | "กรุงไทย เคียงข้างไทย เคียงข้างคุณ" |
| เรียกผู้ใช้ว่า "คุณ" ตลอด | "ให้กรุงไทยช่วยอะไรดีคะ?" |
| ประโยชน์มาก่อนฟีเจอร์ | ผลลัพธ์ขึ้นก่อน รายละเอียดตามหลัง |
| ชวนทำ ไม่บังคับ | "ขอเชิญ…", "ร่วม…" |
| คำเด่นประจำแบรนด์ | "ตอบโจทย์", "ครบจบ", "สะดวก", "คุ้ม" |
| ปุ่มสั้น กริยานำ | "อ่านรายละเอียด", "ดูทั้งหมด" |
| เมนูเป็นคำนามสั้น | "เงินฝาก", "สินเชื่อ", "ลงทุน" |

### โทนปัจจุบันของเว็บ (จุดที่ควรปรับ)
- ชื่อ "ปักหลัก เพื่อก้าวต่อ" ดีอยู่แล้ว — เป็นภาษาตระกูลเดียวกับกรุงไทยโดยธรรมชาติ (ให้กำลังใจ, มองไปข้างหน้า) **คงไว้**
- คำอธิบายยังเป็นภาษาระบบ/เทคนิค: "เว็บแอปพลิเคชันสำหรับตรวจสุขภาพทางการเงิน ช่วยวิเคราะห์สถานะหนี้สิน" — เน้นว่าระบบ *ทำอะไร* ไม่ใช่ผู้ใช้ *ได้อะไร*
- หน้าที่เกี่ยวกับหนี้ต้องระวังเป็นพิเศษ: ห้ามใช้ภาษาตัดสิน/ตีตรา (เช่น "ติดหนี้", "มีปัญหา") → ใช้ภาษา "เดินหน้าต่อ/จัดการได้" แบบกรุงไทย

### สูตรแปลงภาษา (ใช้กับทุกข้อความ)
1. ระบบทำ X → **คุณได้ Y** ("วิเคราะห์สถานะหนี้สิน" → "รู้สถานะการเงินของคุณใน 3 นาที")
2. เติมความรู้สึก "เคียงข้าง" ("กรอกข้อมูล" → "เล่าให้เราฟัง แล้วให้เราช่วยดูแลต่อ")
3. ปุ่ม = กริยา + ผลลัพธ์ ไม่เกิน 4 คำ
4. เลี่ยงคำลบ: หนี้เสีย/ผิดนัด → "ภาระที่จัดการได้", ปัญหา → "โจทย์"
5. ปิดทุกหน้าด้วยทางไปต่อเสมอ (CTA หรือช่องทางติดต่อ)

## 2. ข้อความสำเร็จรูปรายหน้า (พร้อมใช้)

### 2.1 หน้าแรก (Landing / Hero)

| ตำแหน่ง | เดิม (แนวปัจจุบัน) | ใหม่ (โทนกรุงไทย) |
|---|---|---|
| Headline | ปักหลัก เพื่อก้าวต่อ | **ปักหลัก เพื่อก้าวต่อ** *(คงไว้ — เพิ่ม tagline ใต้ชื่อ)* |
| Tagline | เว็บแอปพลิเคชันสำหรับตรวจสุขภาพทางการเงิน | **เคียงข้างคุณทุกก้าว สู่การเงินที่มั่นคง** |
| Sub-copy | ช่วยวิเคราะห์สถานะหนี้สิน และรับคำแนะนำแนวทางแก้ไขหนี้ | **เช็กสุขภาพการเงินของคุณฟรี ใน 3 นาที รู้สถานะ พร้อมแนวทางจัดการหนี้ที่ตอบโจทย์ชีวิตคุณ** |
| CTA หลัก | เริ่มทำแบบประเมิน | **เช็กสุขภาพการเงินเลย** |
| CTA รอง | เข้าสู่ระบบ | **เข้าสู่ระบบ** |
| Trust line | — | **ข้อมูลของคุณเป็นความลับ ใช้เพื่อคำแนะนำของคุณเท่านั้น** |

จุดขาย 3 ข้อ (การ์ดใต้ hero):
- **รู้เร็ว** — เช็กสถานะการเงินครบจบในที่เดียว ไม่ต้องเตรียมเอกสาร
- **รู้จริง** — ผลวิเคราะห์เฉพาะของคุณ พร้อมแนวทางที่ทำได้ทันที
- **ไม่เดินคนเดียว** — ทีมงานพร้อมเคียงข้าง ให้คำปรึกษาต่อทุกช่องทาง

### 2.2 หน้าแบบประเมิน (Health Check)

| ตำแหน่ง | ใหม่ (โทนกรุงไทย) |
|---|---|
| หัวข้อ | **เล่าเรื่องการเงินของคุณให้เราฟัง** |
| คำอธิบาย | ตอบสั้นๆ ไม่กี่ข้อ ใช้เวลาไม่เกิน 3 นาที ทุกคำตอบเป็นความลับ |
| Progress | อีกนิดเดียว! เหลืออีก {n} ข้อ |
| ปุ่มถัดไป | **ถัดไป** |
| ปุ่มย้อน | ย้อนกลับ |
| ปุ่มส่ง | **ดูผลของฉัน** |
| ข้อความกำลังโหลด | กำลังวิเคราะห์ให้คุณอยู่ อีกครู่เดียว… |

ตัวอย่างการปรับคำถาม (หลักการ: ถามแบบเพื่อน ไม่ใช่แบบฟอร์มราชการ):
- "ระบุจำนวนหนี้สินคงค้าง" → **"ตอนนี้คุณมีภาระผ่อนต่อเดือนรวมประมาณเท่าไหร่"**
- "ท่านมีประวัติผิดนัดชำระหรือไม่" → **"ช่วงที่ผ่านมา มีงวดไหนที่จ่ายไม่ทันบ้างไหม (ไม่เป็นไร เล่าได้เลย)"**

### 2.3 หน้าผลลัพธ์ (Result)

โครงข้อความตามระดับผล — ทุกระดับต้องจบด้วย "ก้าวต่อไป" เสมอ:

**ระดับดี:**
> **การเงินของคุณแข็งแรงดี เยี่ยมมาก!**
> รักษาจังหวะนี้ไว้ แล้วให้เงินทำงานต่อ — ดูทางเลือกออมและลงทุนที่ตอบโจทย์คุณ
> ปุ่ม: **ดูคำแนะนำสำหรับฉัน**

**ระดับเฝ้าระวัง:**
> **ยังจัดการได้ และคุณเริ่มก่อนใครแล้ว**
> ภาระตอนนี้เริ่มตึงตัว แต่ยังมีทางปรับอีกหลายทาง เราสรุปแนวทางที่เหมาะกับคุณไว้ให้แล้ว
> ปุ่ม: **ดูแนวทางของฉัน**

**ระดับต้องรีบดูแล:**
> **ทุกปัญหามีทางออก และคุณไม่ต้องเดินคนเดียว**
> สถานะตอนนี้ควรได้รับการดูแลโดยเร็ว — คุยกับทีมงานของเราได้ฟรี ไม่มีค่าใช้จ่าย ข้อมูลเป็นความลับ
> ปุ่ม: **ปรึกษาผู้เชี่ยวชาญฟรี**

ส่วนท้ายทุกระดับ: **"ให้เราช่วยอะไรต่อดี?"** + ช่องทางติดต่อ (ล้อ "ให้กรุงไทยช่วยอะไรดีคะ?")

### 2.4 สมัคร / เข้าสู่ระบบ / รหัสผ่าน

| จุด | ข้อความใหม่ |
|---|---|
| หัวสมัครสมาชิก | **เริ่มต้นก้าวแรกกับเรา** — สมัครฟรี ใช้เวลาไม่ถึงนาที |
| หัวเข้าสู่ระบบ | **ยินดีต้อนรับกลับมา** |
| ลืมรหัสผ่าน | ลืมรหัสผ่าน? ไม่เป็นไร กดที่นี่แล้วเราจะช่วยตั้งใหม่ให้ |
| ปุ่มสมัคร | **สมัครใช้งานฟรี** |
| ปุ่มรีเซ็ต | **ตั้งรหัสผ่านใหม่** |
| สำเร็จ | เรียบร้อย! พร้อมใช้งานแล้ว |

### 2.5 ข้อความระบบ / Error (สำคัญ — จุดที่เสียโทนง่ายที่สุด)

| สถานการณ์ | ❌ ภาษาระบบ | ✅ โทนกรุงไทย |
|---|---|---|
| กรอกไม่ครบ | กรุณากรอกข้อมูลให้ครบถ้วน | เกือบครบแล้ว — ช่วยเติมช่องที่เหลืออีกนิดนะ |
| รหัสผิด | Invalid credentials | อีเมลหรือรหัสผ่านไม่ตรง ลองอีกครั้ง หรือกด "ลืมรหัสผ่าน" ได้เลย |
| ระบบขัดข้อง | เกิดข้อผิดพลาด (Error 500) | ขออภัย ระบบติดขัดชั่วคราว เรากำลังเร่งแก้ไข — ลองใหม่อีกครู่นะ |
| หมดเวลา | Session expired | เพื่อความปลอดภัยของคุณ ระบบพักการใช้งานให้อัตโนมัติ เข้าสู่ระบบอีกครั้งได้เลย |

### 2.6 Privacy / ความเป็นส่วนตัว
หัวข้อ: **ข้อมูลของคุณ คือความลับของเรา**
ประโยคเปิด: "เราเก็บข้อมูลเท่าที่จำเป็น ใช้เพื่อให้คำแนะนำที่เหมาะกับคุณเท่านั้น และไม่ส่งต่อให้ใครโดยไม่ได้รับอนุญาตจากคุณ"

## 3. กติกาภาษาประจำเว็บ (Voice Guideline สรุป 1 หน้า)

**เราคือ**: เพื่อนที่รู้เรื่องการเงิน ยืนข้างๆ ตอนคุณตั้งหลัก
**เราไม่ใช่**: เจ้าหนี้, ครู, แบบฟอร์มราชการ

| กติกา | ทำ | ไม่ทำ |
|---|---|---|
| สรรพนาม | "คุณ" + "เรา" | "ท่าน", "ผู้ใช้งาน" |
| ประโยค | สั้น กระชับ ประโยชน์นำ | ประโยคซ้อนยาว ศัพท์เทคนิค |
| อารมณ์ | ให้กำลังใจ มีทางออกเสมอ | ตีตรา ขู่ เร่งรัด |
| ปุ่ม | กริยา + ผลลัพธ์ ≤4 คำ | "Submit", "ตกลง" เปล่าๆ |
| ตัวเลข | เจาะจง ("3 นาที", "ฟรี") | คลุมเครือ ("รวดเร็ว", "ง่ายดาย") |
| คำแบรนด์ | เคียงข้าง, ก้าวต่อ, ตั้งหลัก, ตอบโจทย์, ครบจบ | หนี้เสีย, ผิดนัด, ปัญหาหนี้สิน (ในหน้า user) |

**เช็กก่อนปล่อยทุกข้อความ 3 ข้อ:**
1. อ่านแล้วรู้สึกว่า "มีคนอยู่ข้างเรา" ไหม
2. ผู้ใช้รู้ไหมว่าได้อะไร และต้องกดอะไรต่อ
3. มีคำไหนทำให้คนเป็นหนี้รู้สึกถูกตัดสินหรือเปล่า
